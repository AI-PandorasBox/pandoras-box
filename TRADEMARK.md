## What is covered

"Pandora's Box" (word mark) and any Pandora's Box logo or wordmark imagery published on `ai-pandorasbox.co.uk` or in this repository are trademarks of the project's intended IP-holding entity (Pandora's Box Holdings Ltd, pending formation).

These trademarks are NOT covered by the Apache 2.0 licence that governs the source code. Apache 2.0 explicitly excludes trademark rights from the licence grant (see Section 6 of `LICENSE`).

## What you can do without permission

- Run the software for any purpose, including commercial use.
- Modify and redistribute the source code under Apache 2.0.
- State that your project is "built on Pandora's Box" or "powered by Pandora's Box" in factual descriptions, release notes, or documentation.
- Use the Pandora's Box name in academic papers, blog posts, articles, and tutorials describing the software, in line with normal fair-use conventions.

## What requires permission

- Distribute a modified version of the software under the name "Pandora's Box" or any confusingly similar name. (Use a different product name for forks.)
- Use the Pandora's Box logo, wordmark imagery, or hero artwork in your own product branding, app icons, or marketing materials.
- Sell goods or services labelled "Pandora's Box" or trade off the project's name in a way that implies endorsement.
- Register a domain, social media account, or company name that incorporates "Pandora's Box" or a confusingly similar mark.

## How to request permission

Open a GitHub Discussion describing the intended use. We respond on a case-by-case basis. Permission is generally granted for non-commercial, non-confusing, attributed uses; commercial uses require a written licence.

## Enforcement

The project's IP-holding entity reserves the right to enforce these marks to prevent confusion in the marketplace and to protect the project's reputation. Enforcement is intentional and consistent; we won't selectively target some uses while ignoring others.

## Reporting infringement

If you believe a third party is misusing the Pandora's Box name or logo in a way that creates user confusion or implies endorsement, open a GitHub Issue with the relevant details. We will investigate.

---

*This trademark policy is intentionally narrower than commercial trademark policies seen in other open-source projects. It permits factual descriptive use freely while reserving distinctive-mark uses to the project. Once the holding entity is formed and the marks are registered, this document will be updated to reflect the registered owner, registration numbers, and any geographic limitations.*
