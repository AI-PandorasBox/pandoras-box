# vector-kb requirements

- **Node 22+** (built-in `node:sqlite`).
- **ollama** module installed and running, with an embedding model pulled
  (default `nomic-embed-text`: `ollama pull nomic-embed-text`).
- ~Disk: vectors are small; a few thousand memories is a few MB.

No third-party accounts, API keys, or network egress. Cost: free (local).
