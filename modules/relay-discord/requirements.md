# relay-discord -- Requirements

| Requirement | Value |
|-------------|-------|
| Discord account | Required |
| Discord server | Bot must be invited |
| Bot token | From discord.com/developers/applications |
| Channel ID | Right-click channel -> Copy Channel ID (Developer Mode must be on) |
| Permissions | Send Messages, Read Message History, View Channels |
| Intents | Message Content Intent, Server Members Intent |
