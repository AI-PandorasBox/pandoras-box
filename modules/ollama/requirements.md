# ollama -- Requirements

| Requirement | Value |
|-------------|-------|
| RAM | 16 GB recommended (12 GB minimum for gemma3:12b) |
| Disk | ~10 GB for gemma3:12b model |
| Ollama | brew install ollama |
| Model | gemma3:12b (default); gemma3:4b for 8 GB Mac |
| Network | Internet required for initial model download |

## Ports

Ollama listens on `localhost:11434` by default (not exposed to network).
