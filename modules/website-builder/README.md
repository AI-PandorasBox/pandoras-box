# website-builder

> **Website Builder**

**Status:** Optional
**Depends on:** core

## What It Does

Website builder and deployment pipeline. Generates and publishes websites based on
content and design briefs.

## Current Status

This module is documented but not yet implemented. The build instructions are planned
for a future phase.

To request implementation, create a project:
```
bash ZEUS/scripts/zeus-project-create.sh "Implement website-builder module"
```

## Planned Capabilities

- Generate website content from a brief
- Build static sites (HTML/CSS/JS)
- Deploy to hosting services (GitHub Pages, Netlify, etc.)
- Update and republish on instruction

## Requirements (planned)

- Node.js 20 or later
- A hosting account (GitHub Pages, Netlify, Cloudflare Pages, or similar)
- Domain name (optional)
