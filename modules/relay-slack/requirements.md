# relay-slack -- Requirements

| Requirement | Value |
|-------------|-------|
| Slack workspace | Required |
| Slack app | Created at api.slack.com/apps |
| Bot token | xoxb-... (Bot User OAuth Token) |
| App token | xapp-... (App-Level Token for Socket Mode) |
| Scopes | chat:write, channels:history, channels:read, im:history, im:write |
| Socket Mode | Must be enabled (no public webhook required) |
