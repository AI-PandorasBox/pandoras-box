# Audio Narrate

Generates narrated audio from text using a TTS provider (ElevenLabs by default). Optional; inert until a provider key is configured.

## Provides
- `audio-narrate`
- `tts`

## Requires
- `core`

## Configuration
- `ELEVENLABS_API_KEY` — ElevenLabs API key (or other TTS provider key)

## Enabling
Optional module. Ships disabled; enable per-agent in the activation matrix.
