# admin-shell

> **CLI Admin Tools**

**Status:** Optional
**Depends on:** core, docs-server

## What It Does

Creates a Chrome desktop app for the Admin Shell. Opens as a standalone window
(not in a browser tab) for a focused administration experience.

## Requirements

- Google Chrome installed

## How to Open

Double-click "Admin Shell" in ~/Applications/, or run:
```
open ~/Applications/Admin\ Shell.app
```

## Monthly Cost

None.

## Uninstall

```
rm -rf ~/Applications/Admin\ Shell.app
rm -rf ~/.config/pbox-admin-shell
```
