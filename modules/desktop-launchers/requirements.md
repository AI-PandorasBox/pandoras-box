# desktop-launchers -- Requirements

## System Requirements

| Requirement | Minimum |
|-------------|---------|
| macOS | 14.0 |
| `osacompile` | Built-in to macOS -- no install needed |

## Required Credentials

None.

## Permissions

- Each `.app` is owned by your user account (chmod 755)
- macOS may prompt the first time you click ("Are you sure you want to open
  this?") -- click Open
