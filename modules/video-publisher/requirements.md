# video-publisher -- Requirements

| Requirement | Value |
|-------------|-------|
| core | Required |
| ffmpeg | brew install ffmpeg |
| ElevenLabs API key | For voice synthesis |
| ElevenLabs voice ID | From elevenlabs.io voice library |
| YouTube Data API v3 | From Google Cloud Console |
| YouTube OAuth credentials | Client ID and secret (Desktop type) |
| YouTube Channel ID | From YouTube Studio settings |
