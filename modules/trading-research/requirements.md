# trading-research -- Requirements

| Requirement | Value |
|-------------|-------|
| Python 3 | brew install python3 |
| pandas, numpy | pip3 install pandas numpy |
| yfinance | pip3 install yfinance |
| Brokerage API | Credentials from your brokerage |
| Watchlist | JSON file at /opt/pandoras-box/trading-research/watchlist.json |
